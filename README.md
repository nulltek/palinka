# Pálinka ügyfél nyilvántartó

Egyszerű webes ügyfél- és főzésnyilvántartó pálinkafőzdéhez.

## Indítás helyben

```bash
python app.py
```

Megnyitás:

```text
http://127.0.0.1:5177
```

Jelszó:

```text
1961
```

## Render

Start command:

```bash
python app.py
```

Render automatikusan ad `PORT` környezeti változót, az app ezt használja.
